# file-metadata

-It is short microservice which give details about uploaded file.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Harsh-Dobariya/file-metadata.git
```

Go to the project directory

```bash
  cd file-metadata
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

**Client:** HTML, CSS, Javascript
